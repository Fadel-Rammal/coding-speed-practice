# coding-speed-practice


A website for practicing on coding typing speed 

### Features:

- Practice with default Python code.
- Upload your own code from any language or even normal text.
- Calculate wpm, mistakes, and accuracy.

### Technologies used:

- HTML/CSS
- JavaScript
